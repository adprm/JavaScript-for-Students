# JavaScript-Math

A Math object is an object that contains mathematical functions<br>
and several constants to do mathematical calculations such as sin,<br>
cos, tan, exponent, square root, etc.

# Basic writing Math

```
var a = Math.sin(10);
var b = Math.cos(10);
var c = Math.tan(10);
```
