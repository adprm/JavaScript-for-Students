# JavaScript Writing

1. Tag Script

```
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JavaScript Writing</title>
    <script>
        console.log("Learn JavaScript");
    </script>
</head>
<body>
    <script>
        document.write("Hello JavaScript!");
    </script>
</body>
</html>
```

2. External Writing
   `index.html`

```
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>JavaScript Writing</title>
</head>
<body>

</body>

<script src="index.js"></script>
</html>
```

`index.js`

```
alert("This is JavaScript");
```

3. Attribute Writing

`<button onclick="alert('Hello JavaScript')">Click ME!</button>`
